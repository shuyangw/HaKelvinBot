## Getting Started

This project was developed, build and tested with Visual Studio 2019 Community running .NET Core 3.1.

### Prerequisites
 A number of NuGet packages are used: 

Discord.NET
YamlDotNet

They can be installed with the "Manage NuGet packages..." option when right-clicking on the HaKelvinBot project.
### Installation

1. Follow the instructions here: [https://discord.com/developers/docs/intro](https://discord.com/developers/docs/intro) and obtain an API key/
2. Clone the repo
```sh
git clone https://github.com/shuyangw/HaKelvinBot.git
```
3. Install Visual Studio 2019 and required NuGet packages.
4. Enter your API key in `config.yml.dist`
```
key: --key goes here--
```
